# Module Aims and Outcomes:

## Key Aims:

- to introduce you to a range of concepts and principles regarding web, mobile and cloud technologies so as to provide a framework of understanding that will enable you to keep abreast of future developments
- to provide you with knowledge and understanding of some of the techniques, applications and issues relevant to employing web, mobile and cloud technologies, both individually and in combination
- to develop your research, analytical, critical and creative skills in thinking about and discussing web, mobile and cloud technologies
- to develop your communication skills through reading and writing documents related to web, mobile and cloud technologies and using diagrams and design tools
- to facilitate your further study in technological subjects.

# Blocks:

## Block 1 - Web Foundations:

- historic development of the internet and openness
- markup (HTML, XHTML, CSS, XML)
- the definition of an ‘architecture’ and basic client-server architecture
- how reality can be digitally modelled
- how services on the web can be connected to each other
- how services can be scaled to deal with large numbers of concurrent clients
- web application frameworks
- accessibility
- security (encryption, authentication, validation).

## Block 2 - Mobile Applications:

- developing and deploying mobile apps
- the mobile market
- using web services
- interface design
- mobile toolkits.

## Block 3 - Cloud Technology:

- cloud model and components (compute, database, storage, networking)
- different levels of provision (machine, operating system, services)
- different types of cloud (private, public, hybrid)
- implementations (OpenStack, AWS, Microsoft Azure, etc.)
- deploying, monitoring and destroying cloud components
- cloud automation facilities such as automated scaling of applications when there is increased demand.
- Difference between the marketing of cloud and what cloud is actually useful for.
