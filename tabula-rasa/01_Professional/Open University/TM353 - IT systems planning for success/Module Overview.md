# TM353 Module Overview

## Module Basics
- **Full Name:** IT systems: planning for success
- **Credits:** 30 (Stage 3)
- **Focus:** Systems thinking, software engineering, and management for IT systems success
- **No programming required** (basic math in Block 2 Part 3 only)

## Learning Outcomes
- Understand success/failure in IT systems
- Analyse sociotechnical nature (hardware, software, organizational, human components)
- Navigate legal, social, ethical, professional contexts
- Apply project management, maintenance, and evolution techniques

## Prerequisites
- Stage 3 module - assumes higher education experience in related subject
- Helpful (not required): basic awareness of project management and software engineering
- Computer skills: word processing, file management, basic calculations
