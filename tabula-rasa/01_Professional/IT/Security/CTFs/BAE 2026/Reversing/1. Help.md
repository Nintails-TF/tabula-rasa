## Getting Started
- Run `strings` on the binary
- Run `file` on the binary
- Google for how "How do I reverse..."

### Interpreted Executables
When reversing interpreted languages like Java, .NET, Python, etc. the code will likely reverse relatively cleanly. If obfuscation has been applied, start by looking for standard API function calls. Calling these is quite difficult and is a good place to start reversing. Obfuscated variable names can be renamed based on how what they are used for in standard API calls.

### Native Executables
Be aware of what type of binary you have, often elf or PE. Be aware of what platform the binary is for (Linux vs Windows) and the differences between the two. In particular, how threading/forking is done for each. Be aware of different calling conventions and what registers may be used in each.
