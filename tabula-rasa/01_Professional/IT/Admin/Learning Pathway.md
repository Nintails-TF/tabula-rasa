# Fundamentals:

- **TCP/IP** — How data actually moves across networks. Subnetting, routing, ports, protocols. Everything else is just an abstraction on top of this.
- **DNS** — It's always DNS. Understanding resolution, record types, caching, and troubleshooting DNS issues will save you more times than you can count.
- **Linux** — Command line fluency. Navigating, reading logs, managing services, editing configs. The cloud runs on Linux.
- **Windows Server + Active Directory** — Still dominates enterprise environments. User management, group policy, domain services.
- **Firewalls and security thinking** — Understanding traffic flow, access control, and the "default deny" mindset. The vendor doesn't matter, the logic does.
- **Scripting** — Python, PowerShell, or Bash. Not software engineering, just enough to automate repetitive tasks and parse data. This is the single biggest accelerator from mid-level to senior.
	- Automate everything worthwhile.
- **Cloud fundamentals** — Pick one provider (AWS is the safest bet, Azure if your path is Microsoft-heavy). Learn VPCs, compute, storage, IAM, and infrastructure as code (Terraform). 

## Bonus:

- **Troubleshooting methodology** — being the person who finds the root cause fast is worth more than any certification.
- **Documentation** — the person who knows how the environment actually works and can prove it becomes irreplaceable.
- **Automation mindset** — not just scripting ability, but the habit of refusing to do the same thing manually twice.
