# The Idea:

Be a John. Be the person who keeps the lights on and the system running. Achieve this by working with less complex systems first, monitoring them, keeping them healthy, and fixing them. Then increasing complexity over time.

IT Support -> NOC/System/Network/DB Junior Admin -> Mid-level admin -> Senior/Cloud/DevOps/SRE etc -> ?

You are tending a garden, not running a race. The end goal being after work - Is the service running or not?
